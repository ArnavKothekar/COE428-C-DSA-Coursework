#COE428: Algorithms and Data Structures – C Coursework (Winter 2026)

This repository contains laboratory work completed as part of COE428: Algorithms and Data Structures, a second-year Computer Engineering course taken in Winter 2026.
All exercises were implemented in C, with a focus on algorithmic efficiency and memory management, including asymptotic complexity analysis,
divide-and-conquer paradigms, priority queues, and the implementation of abstract data structures such as Heaps and Binary Search Trees. 
The work posted here represents my personal implementations and learning outcomes from the course. It is shared purely to showcase my practical understanding of Algorithms and Data Structures principles. 

Disclaimer: This repository is provided for demonstration and portfolio purposes only. It is not intended for academic use or submission, and should not be copied or used to complete coursework at any institution.
